Golf empty state
Switch sports above. Until Links Cup 2026 data lands, the Golf tab renders the same shell with an empty-state hero ("First round tees off [date]") and a faded schedule strip. Same components, played: false everywhere.
Theming
Accent color is the only thing that changes between sports — crimson for poker, green for golf. Driven by [data-sport] on <body>. Layout, type, components are identical.
