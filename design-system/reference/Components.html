Components

One kit, two faces

Every component is themed via .theme-links or .theme-felt. Side-by-side previews show both. Markup is identical — only the wrapper changes.

01 / Action

Buttons

Primary action is the theme’s green. Secondary is ghost on a tonal surface. Destructive uses the danger token. Plaque buttons are reserved for ceremonial actions on the homepage (“Enter the cup”).

  • Active: 0px + shadow-xs
  • Disabled: opacity 0.4, cursor not-allowed
02 / Input

Forms

Inputs are 48px tall, 1px rule, focus ring uses the theme’s primary. Selects use a custom chevron. Labels are eyebrow-style for legibility on both themes.
